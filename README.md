# MessengerApp

**MessengerApp** — это корпоративный мессенджер с поддержкой обмена сообщениями в реальном времени, авторизацией. Приложение разработано с использованием архитектуры MVVM и интегрировано с Firebase для аутентификации, хранения данных и отправки уведомлений. Работа в рамках курсовой работы

## 🚀 Основные функции

- **Регистрация и авторизация**: Поддержка входа с помощью Firebase Authentication.
- **Чаты в реальном времени**: Отправка и получение сообщений с использованием Firestore.
- В БУДУЩЕМ:
- **Push-уведомления**: Оповещения о новых сообщениях через Firebase Cloud Messaging.
- **Профили пользователей**: Возможность редактирования профиля пользователя.
- **Поддержка оффлайн-режима**: Сообщения сохраняются и синхронизируются при восстановлении соединения.

---

## 🛠️ Используемые технологии и библиотеки

- **Kotlin** — основной язык разработки.
- **MVVM** — архитектура приложения.
- **Firebase Authentication** — авторизация пользователей.
- **Firebase Firestore** — хранение данных (чаты и профили пользователей).
- **Firebase Cloud Messaging** — отправка push-уведомлений.
- **LiveData и ViewModel** — реактивный UI.
- **Coroutines** — асинхронные операции.
- **Glide** — загрузка и кэширование изображений профиля.

---

## 📐 Архитектура

Приложение разработано на основе паттерна **MVVM**:
- **Model** — управляет данными и взаимодействует с Firebase для отправки и получения сообщений.
- **View** — отображает данные и подписывается на обновления через `LiveData`.
- **ViewModel** — обеспечивает связь между Model и View, управляет состоянием и логикой приложения.
